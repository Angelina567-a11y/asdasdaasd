# Random Task Generator

Автор: Иван Иванов

Описание:
Это GUI-приложение позволяет случайным образом генерировать задачи из предопределенного списка, сохранять историю выполненных задач, фильтровать задачи по типу и добавлять новые. Всё сохраняется в JSON-файл для сохранения данных между сессиями.

Использование:
- Нажмите «Сгенерировать задачу» для получения случайной задачи.
- Используйте фильтр для отображения задач по типам: учёба, спорт, работа, отдых.
- Добавляйте новые задачи через поля и кнопку «Добавить задачу».
- История отображается в списке.

Требования:
- Python 3.x
- Стандартные библиотеки (tkinter, json, random)

Запуск:
1. Откройте терминал и перейдите в папку проекта.
2. Запустите `python main.py`.
3. Используйте приложение по инструкции.
